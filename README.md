## 📌 What this app does

ppt-agent-workflow-san is a desktop tool for gradual, interactive PowerPoint creation. It helps you build slides step by step, so you can review each part before moving on.

Use it to:

- Create a new slide deck from a topic
- Guide the content flow one step at a time
- Review and adjust text before final output
- Keep the slide structure clear and organized

It is built for people who want help making presentation content without starting from a blank file.

## 🧭 How to use it

The app follows a simple guided flow.

### Step 1: Start a new project

Open the app and create a new presentation project. Give it a clear name so you can find it later.

### Step 2: Enter your topic

Type the topic you want the slides to cover. Keep it short and direct. For example:

- Sales plan review
- Product launch summary
- Training overview
- Project status update

### Step 3: Build the slide outline

The app helps you break the topic into parts. You can review the outline and make changes before you move on.

### Step 4: Add slide content

Work through each slide one by one. Check the text, order, and focus of each slide before continuing.

### Step 5: Review and refine

You can go back and edit content if a slide needs a clearer message or shorter text.

### Step 6: Export your deck

When your slides are ready, export the file so you can open it in PowerPoint or share it with others.
